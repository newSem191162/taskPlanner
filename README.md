# 📋 Планировщик Задач — PWA

Интеллектуальный планировщик задач с автоматическим распределением по времени, разбиением на подзадачи и анализом выполнимости.

## 🚀 Быстрый старт

### Установка
```bash
npm install
```

### Запуск
```bash
npm start
```

Откройте http://localhost:3000 в браузере.

### Установка как PWA
1. Откройте приложение в Chrome/Edge/Safari
2. Нажмите "Установить приложение" в адресной строке
3. Приложение появится на рабочем столе

## ✨ Возможности

### 📋 Управление задачами
- Создание, редактирование, удаление задач
- **Приоритизация**: шкала важности 1-10
- **Сложность**: оценка в часах
- **Ресурсы**: учет необходимых материалов
- **Категории**: Работа, Учеба, Личное, Проекты, Здоровье
- **Поиск**: по названию, описанию, категории
- **Фильтры**: все / активные / просроченные / выполненные
- **Сортировка**: по приоритету, сроку, сложности, дате создания

### 🤖 Интеллектуальное планирование
- **Авторазбиение**: автоматическое разбиение на подзадачи по типу:
  - Презентации, отчеты, разработка, дизайн, встречи, статьи, учеба
- **Drag & Drop**: для подзадач
- **Прогресс**: визуальный индикатор выполнения

### 📅 Календарь
- Визуальное распределение задач по дням
- Автоматический расчет часов на каждый день
- Навигация по месяцам
- Просмотр задач на конкретный день

### 📊 Аналитика
- **График загрузки**: на 14 дней вперед
- **Распределение по категориям**
- **Проверка выполнимости**:
  - Просроченные задачи
  - Перегрузка (>8ч/день)
  - Предложения по переносу сроков

### 🎨 Интерфейс
- **Темная/светлая тема**
- **Адаптивный дизайн**: мобильные и десктоп
- **Push-уведомления**: о дедлайнах
- **PWA shortcuts**: быстрый доступ к действиям

### 💾 Данные
- **IndexedDB**: локальное хранение
- **Экспорт/Импорт**: JSON backup
- **Офлайн-режим**: работа без интернета
- **PWA**: установка на устройство

## 🧠 Алгоритм авторазбиения

Приложение анализирует название задачи и автоматически создает подзадачи:

| Тип задачи | Подзадачи |
|------------|-----------|
| **Презентация** | Сбор материалов → Структура → Визуал → Текст → Репетиция |
| **Отчет** | Данные → Структура → Написание → Оформление |
| **Разработка** | Проектирование → Окружение → Код → Тесты → Документация |
| **Дизайн** | Референсы → Наброски → Концепция → Детали → Файлы |
| **Встреча** | Повестка → Согласование → Материалы → Организация |
| **Статья** | Исследование → План → Черновик → Редактура |
| **Учеба** | Объем → Темы → Теория → Практика → Повторение |
| **Другое** | Универсальное разбиение на 2-5 этапов |

## 🔍 Анализ выполнимости

Система проверяет:
1. **Просрочка** — дедлайн уже прошел
2. **Перегрузка** — требуется >8 часов в день
3. **Ресурсы** — недоступны необходимые материалы

При обнаружении проблем:
- 🔴 Просроченные — требуют немедленного внимания
- 🟡 Перегрузка — предлагает оптимальный дедлайн
- 🟢 Ресурсы — напоминает проверить доступность

## 📱 Установка на мобильное устройство

### Android (Chrome)
1. Откройте сайт в Chrome
2. Нажмите ⋮ → "Добавить на главный экран"
3. Приложение установится как нативное

### iOS (Safari)
1. Откройте сайт в Safari
2. Нажмите "Поделиться" → "На экран Домой"
3. Приложение появится на домашнем экране

### Desktop (Chrome/Edge)
1. Нажмите на иконку установки в адресной строке
2. Или ⋮ → "Установить приложение"

## 🛠️ Технологии

- **Frontend**: Vanilla JavaScript, CSS3, HTML5
- **Storage**: IndexedDB (браузерная БД)
- **PWA**: Service Worker, Web App Manifest, Push API, Background Sync
- **Backend**: Node.js + Express (статика)
- **UI**: Кастомный CSS, адаптивный дизайн, CSS Grid/Flexbox
